Colophon

I use the Wintersmith static site generator to manage all content on this site, and a simple bit of python to push it live.

I am tired of using bootstrap or insert new grid layout framework here, it kills my creativity. So I am just writing CSS and markup by hand, although I am using jade as well, for now.

Fonts are from Google Fonts, I really like Cabin and Source Code Pro which I am using along with the excellent Tomorrow Night Theme for syntax highlighting of code.
